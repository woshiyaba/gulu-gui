# 洛克王国精灵图鉴 API 文档

- **服务框架**：FastAPI
- **版本**：`1.0.0`，见 `api/main.py`
- **接口前缀**：公共业务接口统一以 `/api` 开头，健康检查为 `/`
- **数据格式**：请求和响应均为 JSON；列表接口通常返回 `{ total, items }` 或 `{ total, page, page_size, items }`
- **CORS**：当前允许所有来源访问，见 `api/main.py`

本文档只记录前端可直接调用的公共接口。运营后台接口在 `/api/ops` 下，需要 Bearer Token，不在本文档展开。

---

## 通用说明

| 场景 | HTTP 状态 | 响应说明 |
|------|-----------|----------|
| 成功 | `200` | 返回接口对应 JSON 结构 |
| 参数校验失败 | `422` | FastAPI 默认校验错误 |
| 精灵不存在 | `404` | `{"detail":"精灵不存在"}` |
| 阵容不存在或未启用 | `404` | `{"detail":"阵容不存在或未启用"}` |

## 6. 精灵列表

| 项目 | 内容 |
|------|------|
| 方法 / 路径 | `GET /api/pokemon` |
| 响应体 | `PokemonListResponse` |

查询参数：

| 参数 | 类型 | 必填 | 默认 | 说明 |
|------|------|------|------|------|
| `name` | `string` | 否 | `""` | 精灵名称关键词，模糊搜索 |
| `attr` | `string[]` | 否 | `null` | 属性多选筛选，同类条件为 AND；多个值写作 `?attr=火&attr=水` |
| `egg_group` | `string[]` | 否 | `null` | 蛋组多选筛选，同类条件为 AND |

| `page` | `integer` | 否 | `1` | 页码，最小 `1` |
| `page_size` | `integer` | 否 | `30` | 每页数量，范围 `1` 到 `100` |

| 字段 | 类型 | 说明 |

# 属性计算
系数：L = (种族值 + 个体值/2) / 100
努力值默认值：
生命 hp 100
物攻 atk 50
魔攻 matk 50
物防 def_val 50
魔防 mdef 50
速度 spd 50
性格系数 不加不减则为 1
算法均四舍五入展示
生命：（（2L + 1）* 等级 + 50L + 10）* 性格系数 + 努力值
物攻：（L * 等级 + 50L + 10）* 性格系数 + 努力值
魔攻：（L * 等级 + 50L + 10）* 性格系数 + 努力值
物防：（L * 等级 + 50L + 10）* 性格系数 + 努力值
魔防：（L * 等级 + 50L + 10）* 性格系数 + 努力值
速度：（L * 等级 + 50L + 10）* 性格系数 + 努力值


伤害计算公式（PVP）：
(威力 * 攻击 * 37/41) 四舍五入
基础威力：
威力 = （技能原威力 + 威力数值变化）* 本系技能加成（1.25） * 攻击力加成 * 威力加成 * 克制系数（1|2|3）/ 双防加成
非本系技能：(威力 * 攻击 * 37/41) / 防御 => 结果四舍五入
本系技能：(威力 * 1.25 * 攻击 * 37/41) / 防御 => 结果四舍五入

已知威力的情况下：
((威力 * 攻击 * 37/41) / 防御) => 结果四舍五入

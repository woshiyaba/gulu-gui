# 属性计算
系数：L = (种族值 + 个体值/2) / 100
努力值默认值：
生命 hp 100
物攻 atk 50
魔攻 matk 50
物防 def_val 50
魔防 mdef 50
速度 spd 50
性格系数 不加不减则为 1
算法均四舍五入展示
生命：（（2L + 1）* 等级 + 50L + 10）* 性格系数 + 努力值
物攻：（L * 等级 + 50L + 10）* 性格系数 + 努力值
魔攻：（L * 等级 + 50L + 10）* 性格系数 + 努力值
物防：（L * 等级 + 50L + 10）* 性格系数 + 努力值
魔防：（L * 等级 + 50L + 10）* 性格系数 + 努力值
速度：（L * 等级 + 50L + 10）* 性格系数 + 努力值


伤害计算公式：
(威力 * 攻击 * 37/41) 四舍五入
(威力 * 攻击 * 37/41) / 防御 => 结果向下取整

# 属性计算
系数：L = (种族值 + 个体值/2) / 100
努力值默认值：
生命 hp 100
物攻 atk 50
魔攻 matk 50
物防 def_val 50
魔防 mdef 50
速度 spd 50
性格系数 不加不减则为 1
算法均四舍五入展示
生命：（（2L + 1）* 等级 + 50L + 10）* 性格系数 + 努力值
物攻：（L * 等级 + 50L + 10）* 性格系数 + 努力值
魔攻：（L * 等级 + 50L + 10）* 性格系数 + 努力值
物防：（L * 等级 + 50L + 10）* 性格系数 + 努力值
魔防：（L * 等级 + 50L + 10）* 性格系数 + 努力值
速度：（L * 等级 + 50L + 10）* 性格系数 + 努力值


伤害计算公式（PVP）：
(威力 * 攻击 * 37/41) 四舍五入基础威力：
威力 = （技能原威力 + 威力数值变化）* 本系技能加成（1.25） * 攻击力加成 * 威力加成 * 克制系数（0.25|0.5|1|2|3）/ 双防加成
非本系技能：(威力 * 攻击 * 37/41) / 防御 => 结果四舍五入
本系技能：(威力 * 1.25 * 攻击 * 37/41) / 防御 => 结果四舍五入

已知威力的情况下：
((威力 * 攻击 * 37/41) / 防御) => 结果四舍五入

## 1.添加接口，计算宠物技能伤害的具体数值

- 接口会上传 进攻方和被进攻方宠物的 生命，物攻，魔攻，物防，魔防，速度
- 连击次数
- 需要有一个参数判断是已知威力 还是基础威力计算
- 还需要知道技能是魔攻还是物攻，通过 "魔攻" | "物攻"
- 如果是已威力技能 直接根据 ((威力 * 攻击 * 37/41) / 防御) * 连击次数 公式计算伤害 然后进行返回 真实伤害数值

- 如果是基础威力技能计算 需要额外的参数
- 技能的属性
- 进攻方宠物的属性
- 被进攻放宠物的属性列表 因为可能是多属性宠物
- 需要根据技能属性 从attribute.name查询对应的id 然后从 attribute_matchup 中查询attacker_attr_id为技能属性的 然后分别查询 defender_attr_id 为被攻击宠物的属性 对应的克制 attribute_matchup.multiplier 0.5为抵抗 2为克制 1为正常，如果 双属性均被克制 克制系数为*3 如果一个抵抗 一个克制 克制系数为 * 1 如果一个克制 一个正常无变化 则 克制系数为 * 2 同理如果双抵抗 克制系数为 * 0.25 单一抵抗 一个正常 克制系数为 * 0.5
- 如果进攻方的技能属性 正好为自己的属性 则 需要额外 * 1.25的本系技能加成
- 特性攻击力加成 传增加或者减少的部分20% 0.2 =>计算的时候 使用1+0.2 1.2计算
- 特性威力加成 传增加或者减少的部分20% 0.2 =>计算的时候 使用1+0.2 1.2计算
- 威力固定加值 默认 0 
- 其他威力加成（比如力增）传增加或者减少的部分20% 0.2 =>计算的时候 使用1+0.2 1.2计算
- 其他攻击力加成（比如力增）传增加或者减少的部分20% 0.2 =>计算的时候 使用1+0.2 1.2计算
- 被进攻方 防御加成 40% 则算的时候 使用 1.4计算 -40% 则使用 0.6计算
- 威力 = （技能原威力 + 威力数值变化）* 本系技能加成（1.25） * 攻击力加成 * 威力加成 * 克制系数（0.25|0.5|1|2|3）/ 双防加成 通过这个公式计算威力之后
- ((威力 * 攻击 * 37/41) / 防御) * 连击次数获取最后的伤害结果
